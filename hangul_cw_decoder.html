한글 CW 해독기 by DS1TZE

Frequency : Hz   Speed : wpm   Mic Gain : %  

[IMPORTANT] Setting Guide.

1. Check and open DitDah Bar first. It helps setting process.
2. Set target speed ±5wpm.
3. Set target frequency ±50Hz.
4. Set Mic Gain properly. Too high or too low value is not good. When matched, you can see some stick graph pops up in DitDah Bar area.
5. Repeat step 3 & 4 until you can see the stick perfectly matched to signal sound. Focus on the DitDah sticks' length and gap between them.
※ When settings are perfect, it can decode over 30wpm signal.

Intro.


안녕하세요. DS1TZE 입니다.

위 프로그램은 한글 CW 해독기 겸 타건 연습기입니다.
핸드폰이나 노트북 컴퓨터 등 마이크가 있는 장비의 브라우저를 이용해 위 프로그램을 이용하면 됩니다.
입력하려는 신호에 따라 Frequency와 Speed, 그리고 MIC Gain을 조절하면 해독이 더 원활해집니다.
한글 CW 청취 연습기나 Web SDR과 함께 사용해보려면, 컴퓨터의 녹음 장치를 Stereo Mix 정도로 설정한 후에 사용하면 됩니다.
많은 활용 바라며, 건의 사항은 온에어에서 말씀해주시기 바랍니다.
