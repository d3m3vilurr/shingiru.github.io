한글 CW 해독기 by DS1TZE

Signal Speed : wpm  

Intro.


안녕하세요. DS1TZE 입니다.

위 프로그램은 한글 CW 해독기 겸 타건 연습기입니다.
핸드폰이나 노트북 컴퓨터 등 마이크가 있는 장비의 브라우저(반드시 최신 버전 크롬 브라우저)를 이용해 위 프로그램을 이용하면 됩니다.
입력하려는 신호에 따라 Frequency와 Speed, 그리고 MIC Gain을 조절하면 해독이 더 원활해집니다.
-> Tone Frequency와 MIC Gain은 Start 버튼을 누를 때마다 자동으로 처리되니, 리그 앞에 휴대폰을 두고 Start 버튼을 누르기만 하면 됩니다.
DitDah Bar에 나타나는 신호 막대가 신호와 맞을 때까지 들리는 소리의 크기를 조절합니다. 소리가 너무 크면 오히려 해독이 안됩니다.
한글 CW 청취 연습기나 Web SDR, 유투브 영상의 신호를 해독하려면, 컴퓨터의 녹음(소리 입력) 장치를 Stereo Mix* 로 선택하되, 볼륨이 너무 높으면 신호 해석이 잘 안되니 적절히 줄인 후 사용하기 바랍니다.
많은 활용 바라며, 건의 사항은 온에어에서 말씀해주시기 바랍니다.
※ 소리 입력 장치 선택 : Windows 설정 -> 시스템 -> 소리. 만약 입력장치 목록에 Stereo Mix 없으면, [사운드 제어판] -> [녹음]탭에서 Stereo Mix 활성화 후 입력 장치 선택.
